A modern movie ticket management platform developed for the DISM eProject. This full-stack application features a dedicated admin dashboard for theater management and a responsive user interface for movie browsing and ticket booking.

## 🚀 Key Features

* **Real-time Dashboard:** Track total sales, active users, and movie counts at a glance.
* **CRUD Operations:** Full management of movies and cinema venues (Add, Edit, View, Delete).
* **Database Integration:** Dynamic booking system that tracks transactions and ticket classes.
* **Responsive Design:** Optimized for both desktop and mobile viewing with a dark-mode aesthetic.
* **Secure Administration:** Protected admin area for sensitive data management.

## 🛠️ Technology Stack

* **Backend:** PHP (Procedural)
* **Database:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **UI Components:** Custom AdminLTE integration for the dashboard
* **Typography:** Inter & Bebas Neue
